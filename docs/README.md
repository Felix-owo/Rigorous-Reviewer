# Repository Documentation

This folder contains repository-level maintenance documents. The installable
skill package remains in `rigorous-reviewer/`.

```text
docs/
├── compatibility/
│   └── agent_matrix.md
└── routing/
    └── trigger_keyword_support.md
```

- `compatibility/`: host and installer compatibility expectations.
- `routing/`: trigger keyword and multi-skill routing maintenance notes.

Keep user-facing overview content in the root `README.md` and
`README.zh-CN.md`. Keep GitHub-recognized project files such as
`SECURITY.md`, `CONTRIBUTING.md`, and `CHANGELOG.md` at the repository root.

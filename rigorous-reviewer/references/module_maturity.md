# Module Maturity Registry

Use when auditing or extending the skill package. Module maturity is an
engineering signal, not a scientific-evidence score.

## Status definitions

| Status | Meaning |
|---|---|
| Stable | Used by the core workflow or validated by existing scripts/templates. |
| Beta | Implemented and usable, but needs more cross-host examples or fixtures. |
| Draft | Defined as a routing/reference module; use cautiously and verify outputs. |

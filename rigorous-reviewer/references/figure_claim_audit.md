# Figure Claim Audit

For each decisive figure/table/panel:

| Field | Question |
|---|---|
| Defended claim | What claim does this visual support? |
| Necessary data | Which panel/statistic/source data is decisive? |
| Control requirement | What control is required for interpretation? |
| Quantification | Is the plotted quantity sufficient and correctly normalized? |
| Statistics | Is the experimental unit correct? Are uncertainty and multiplicity handled? |
| Alternative explanation | What non-author interpretation remains possible? |
| Source data | Can the result be independently inspected or reproduced? |
| Survival test | Does the central claim survive if this figure is removed? |

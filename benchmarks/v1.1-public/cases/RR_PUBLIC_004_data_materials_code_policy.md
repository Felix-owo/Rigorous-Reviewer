# RR_PUBLIC_004_data_materials_code_policy

## Public-source case card

A high-impact manuscript reports a new experimental and computational workflow. The paper claims that other groups can build on the result, but raw data, processed data, analysis code, materials identifiers, and detailed protocols are missing or only available on request without clear restrictions or timelines.

## Reviewer task

Review whether the availability statement and reproducibility package satisfy high-impact journal expectations for data, code, materials, and protocols.

## Expected reviewer behavior

A rigorous reviewer should require public or controlled-access data availability, executable analysis code or sufficient pseudocode, materials/protocol availability, and a clear statement explaining any legitimate restrictions. The review should link availability failures to whether independent groups can replicate and build on the claims.
